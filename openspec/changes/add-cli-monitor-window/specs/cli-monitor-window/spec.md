## ADDED Requirements
### Requirement: CLI 悬浮监控窗
系统 SHALL 提供独立悬浮监控窗，用于只读展示 tmux 会话列表与终端输出。

#### Scenario: 设置开启后显示悬浮窗
- **WHEN** 用户在设置中开启“显示悬浮窗”并保存
- **THEN** 系统打开或聚焦悬浮监控窗并展示会话列表

#### Scenario: 无会话时展示空状态
- **WHEN** 悬浮监控窗打开且没有可用会话
- **THEN** 系统显示与当前工作区一致的空状态提示

#### Scenario: 只读模式
- **WHEN** 用户在悬浮监控窗中操作终端区域
- **THEN** 系统不向 tmux 发送输入或执行分屏/关闭等快捷操作

#### Scenario: 窗口样式
- **WHEN** 悬浮监控窗创建
- **THEN** 窗口为无边框且允许调整大小

### Requirement: 悬浮窗置顶切换
系统 SHALL 在悬浮监控窗内提供置顶开关，仅影响该悬浮窗自身。

#### Scenario: 开启置顶
- **WHEN** 用户在悬浮监控窗内开启置顶开关
- **THEN** 悬浮监控窗保持在其他窗口之上

#### Scenario: 关闭置顶
- **WHEN** 用户在悬浮监控窗内关闭置顶开关
- **THEN** 悬浮监控窗恢复正常层级显示

### Requirement: 设置控制显示
系统 SHALL 在设置中提供“显示悬浮窗”开关，保存后立即生效。

#### Scenario: 关闭悬浮窗
- **WHEN** 用户关闭“显示悬浮窗”并保存
- **THEN** 系统关闭悬浮监控窗且不影响现有 tmux 会话

#### Scenario: 手动关闭不影响设置
- **WHEN** 用户手动关闭悬浮监控窗
- **THEN** 设置中的“显示悬浮窗”保持开启状态
