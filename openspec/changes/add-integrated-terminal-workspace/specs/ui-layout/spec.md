## ADDED Requirements

### Requirement: 双模式布局

系统应当(SHALL)在项目列表模式与开发模式之间切换布局。

#### Scenario: 项目列表模式布局
- **WHEN** 应用处于项目列表模式
- **THEN** 应当显示项目列表视图
- **AND** 隐藏开发模式工作区

#### Scenario: 开发模式布局
- **WHEN** 应用处于开发模式
- **THEN** 应当显示开发模式工作区
- **AND** 隐藏项目列表视图

### Requirement: 模式切换

系统应当(SHALL)支持在项目列表模式和开发模式之间切换。

#### Scenario: 从项目列表进入开发模式
- **WHEN** 用户触发进入开发模式的操作
- **THEN** 应用应当隐藏项目列表视图
- **AND** 显示工作空间视图
- **AND** 保持应用状态（搜索、筛选等）

#### Scenario: 从开发模式返回项目列表
- **WHEN** 用户点击"返回项目列表"按钮
- **THEN** 应用应当隐藏工作空间视图
- **AND** 显示项目列表视图
- **AND** 保留所有打开的终端会话（后台运行）

#### Scenario: 再次进入开发模式时恢复状态
- **WHEN** 用户从项目列表再次进入开发模式
- **THEN** 系统应当恢复之前打开的所有标签页
- **AND** 恢复最后活跃的标签选择状态
