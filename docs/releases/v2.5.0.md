# DevHaven v2.5.0 Release Notes

- 发布日期：2026-02-11
- Tag：`v2.5.0`
- 对比版本：`v2.4.2` -> `v2.5.0`

## 本次升级内容

### 1. 合并 Term 分支
已将 `Term` 分支合并到 `main`，纳入以下关键变更：
- `feat(terminal): add pane search and local path link opening`（`520c818`）
- `chore(openspec): remove openspec artifacts`（`bbbf4f6`）

### 2. 终端体验增强
- 新增终端 Pane 内搜索能力（支持快速打开/关闭搜索并高亮匹配结果）。
- 终端输出中的本地路径支持识别并打开，提高排障与跳转效率。
- 优化 macOS 下终端快捷键处理，减少与系统默认行为冲突。

### 3. 项目文档与工程整理
- 清理 OpenSpec 历史产物，减少仓库冗余内容。
- README 在本轮更新中已重写并补充截图，便于新用户快速上手。

## 版本升级

本次发布将应用版本统一升级为 `2.5.0`：
- `package.json`
- `package-lock.json`
- `src-tauri/tauri.conf.json`
- `src-tauri/Cargo.toml`
- `src-tauri/Cargo.lock`

## 兼容性说明

- 无新增数据目录或迁移步骤。
- 本地数据仍使用 `~/.devhaven/` 下既有结构。
