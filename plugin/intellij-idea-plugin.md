# DevHaven IntelliJ IDEA 插件

本插件为DevHaven配套的IntelliJ IDEA插件，用于自动同步IntelliJ系列IDE（包括IDEA、WebStorm、PyCharm等）中打开的项目信息到DevHaven应用。

## 主要功能

- 自动检测IntelliJ平台IDE中打开的项目
- 将项目信息同步到DevHaven的项目库
- 提供手动同步命令
